# ðŸ“º Retro Terminal

An old-school terminal interface with CRT monitor effects like scanlines, flicker, and screen glitches.

<Preview
  link="retro-terminal"
  comment={["Retro"]}
  className="bg-card border rounded-xl p-6"
>
  <RetroTerminal 
    welcomeMessage={[
      "RETRO-OS v1.0",
      "Copyright (c) 1985-1995",
      "All rights reserved.",
      "",
      "Type 'help' for available commands."
    ]}
    onCommand={async (cmd) => {
      if (cmd === "help") {
        return [
          "Available commands:",
          "dir - List directory contents",
          "ver - Show system version",
          "date - Display current date",
          "cls - Clear screen"
        ];
      } else if (cmd === "dir") {
        return [
          " Volume in drive C has no label",
          " Volume Serial Number is 1337-CAFE",
          "",
          " Directory of C:\\",
          "",
          "COMMAND  COM     63,588 01-01-95  5:00a",
          "CONFIG   SYS      1,024 01-01-95  5:00a",
          "AUTOEXEC BAT        512 01-01-95  5:00a",
          "SYSTEM   DAT     32,768 01-01-95  5:00a",
          "        4 file(s)     97,892 bytes",
          "        0 dir(s)   1,048,576 bytes free"
        ];
      } else if (cmd === "ver") {
        return "RETRO-OS Version 1.0";
      } else if (cmd === "date") {
        return new Date().toLocaleDateString();
      } else if (cmd === "cls") {
        return "Screen cleared";
      } else {
        return `Bad command or file name: ${cmd}`;
      }
    }}
  />
</Preview>

## ðŸ”§ Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `undefined` | Additional CSS classes |
| `scanlines` | `boolean` | `true` | Enable scanline effect |
| `flicker` | `boolean` | `true` | Enable screen flicker effect |
| `glitch` | `boolean` | `true` | Enable random glitch effects |
| `prompt` | `string` | `C:>` | Command prompt symbol |
| `welcomeMessage` | `string \| string[]` | `RETRO-OS v1.0...` | Initial message shown |
| `onCommand` | `(command: string) => Promise<string \| string[]> \| string \| string[]` | `undefined` | Command handler function |

All props from `BaseTerminal` are also supported.
