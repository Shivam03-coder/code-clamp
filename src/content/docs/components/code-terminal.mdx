## ðŸ”§ Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `undefined` | Additional CSS classes |
| `theme` | `"dark" \| "light"` | `"dark"` | Color theme for the terminal |
| `language` | `"javascript" \| "typescript" \| "html" \| "css" \| "json"` | `"javascript"` | Programming language for syntax highlighting |
| `initialCommands` | `TerminalCommand[]` | `[]` | Pre-populated code examples |
| `onCommand` | `(command: string) => Promise<string \| string[]> \| string \| string[]` | `undefined` | Command handler function |

All props from `BaseTerminal` are also supported.

## ðŸŽ¨ Supported Languages

The Code Terminal supports syntax highlighting for:

- JavaScript
- TypeScript
- HTML
- CSS
- JSON

Each language has its own prompt prefix and specialized highlighting rules.

## ðŸŒ“ Themes

Two built-in themes are available:

- `dark` - Dark background with light text (default)
- `light` - Light background with dark text

\`\`\`tsx
// Dark theme (default)
<CodeTerminal theme="dark" />

// Light theme
<CodeTerminal theme="light" />
\`\`\`

The Code Terminal is perfect for code demonstrations, tutorials, or creating interactive coding playgrounds.
