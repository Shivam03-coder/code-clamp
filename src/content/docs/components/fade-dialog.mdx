## ðŸ”§ Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `React.ReactNode` | `undefined` | Dialog title |
| `description` | `React.ReactNode` | `undefined` | Dialog description |
| `children` | `React.ReactNode` | `required` | Dialog content |
| `triggerText` | `string` | `"Open Dialog"` | Text for the trigger button |
| `confirmText` | `string` | `"Confirm"` | Text for the confirm button |
| `cancelText` | `string` | `"Cancel"` | Text for the cancel button |
| `onConfirm` | `() => void` | `undefined` | Function called when confirm button is clicked |
| `showFooter` | `boolean` | `true` | Whether to show the dialog footer with buttons |
| `className` | `string` | `undefined` | Additional CSS classes for the dialog |
| `duration` | `number` | `0.2` | Animation duration in seconds |
