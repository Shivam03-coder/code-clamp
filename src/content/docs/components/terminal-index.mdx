## ‚ú® Component Variants

1. **Base Terminal**
   - Clean, minimal terminal interface with animated typing cursor
   - Command history with keyboard navigation
   - Animated command responses
   - Customizable prompt and welcome message

2. **Matrix Terminal**
   - Features the iconic Matrix falling code animation in the background
   - Canvas-based animation that adapts to container size
   - Customizable animation density and speed
   - Perfect for cybersecurity or hacker-themed interfaces

3. **Retro Terminal**
   - Old-school CRT monitor effects with scanlines and screen curvature
   - Animated flicker and random glitch effects
   - Authentic vintage computing experience
   - Ideal for retro-themed applications or games

4. **Code Terminal**
   - Syntax highlighting for code snippets
   - Support for multiple programming languages
   - Light and dark themes
   - Perfect for code demonstrations or tutorials

5. **Interactive Terminal**
   - Clickable command buttons for easier interaction
   - Animated command panel that can be toggled
   - Command tooltips with descriptions
   - Ideal for guided demos or user-friendly interfaces

## üß© Key Animation Features

- **Typing Cursor**: Animated blinking cursor with smooth transitions  
- **Command Execution**: Animated appearance of command responses  
- **Welcome Message**: Staggered animation for multi-line welcome messages  
- **Visual Effects**: CRT screen effects, Matrix rain, and glitch animations  
- **Interactive Elements**: Smooth transitions for buttons and panels
