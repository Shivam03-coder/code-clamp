## ðŸ”§ Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `undefined` | Additional CSS classes |
| `scanlines` | `boolean` | `true` | Enable scanline effect |
| `flicker` | `boolean` | `true` | Enable screen flicker effect |
| `glitch` | `boolean` | `true` | Enable random glitch effects |
| `prompt` | `string` | `C:>` | Command prompt symbol |
| `welcomeMessage` | `string \| string[]` | `RETRO-OS v1.0...` | Initial message shown |
| `onCommand` | `(command: string) => Promise<string \| string[]> \| string \| string[]` | `undefined` | Command handler function |

All props from `BaseTerminal` are also supported.
