A terminal interface with a "Matrix-style" falling code animation in the background.

<Preview
  link="matrix-terminal"
  comment={["Animation"]}

      "Matrix Terminal Initialized",

    onCommand={async (cmd) => {
      if (cmd === "connect") {
        return [
          "Establishing secure connection...",
          "Access granted.",
          "Welcome to the Matrix."
        ];

        return `Unknown command: ${cmd}`;

## ðŸ”§ Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `undefined` | Additional CSS classes |
| `matrixDensity` | `number` | `50` | Density of falling characters |
| `matrixSpeed` | `number` | `1` | Speed of animation |
| `prompt` | `string` | `>` | Command prompt symbol |
| `welcomeMessage` | `string \| string[]` | `Matrix Terminal Initialized...` | Initial message shown |
| `onCommand` | `(command: string) => Promise<string \| string[]> \| string \| string[]` | `undefined` | Command handler function |

All props from `BaseTerminal` are also supported.
