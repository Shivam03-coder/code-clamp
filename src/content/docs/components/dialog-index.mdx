## Installation

All dialog components are available in the `@/components/ui/animated-dialog` directory. To use them, you'll need to install the required dependencies:

\`\`\`bash
# Install Framer Motion
npm install framer-motion
# or
yarn add framer-motion
# or
bun add framer-motion
\`\`\`

## Usage

Import the desired dialog component:

{/* \`\`\`tsx
import { FadeDialog } from "@/components/ui/animated-dialog";
// or
import { ScaleDialog } from "@/components/ui/animated-dialog";
// or
import { SlideDialog } from "@/components/ui/animated-dialog";
// or
import { FlipDialog } from "@/components/ui/animated-dialog";
// or
import { DrawerDialog } from "@/components/ui/animated-dialog";
\`\`\` */}

Then use it in your component:

{/* \`\`\`tsx
export default function MyComponent() {
  return (
    <FadeDialog
      title="Dialog Title"
      description="Dialog description goes here."
      triggerText="Open Dialog"
      onConfirm={() => {
        // Handle confirmation
        console.log("Confirmed!");
      }}
    >
      <p>Your dialog content goes here.</p>
    </FadeDialog>
  );
}
\`\`\` */}

For more detailed documentation on each component, visit their individual pages:
