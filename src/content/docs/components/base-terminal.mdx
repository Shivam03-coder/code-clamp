## üîß Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `undefined` | Additional CSS classes |
| `prompt` | `string` | `$` | Command prompt symbol |
| `welcomeMessage` | `string \| string[]` | `Welcome to Terminal...` | Initial message shown |
| `initialCommands` | `TerminalCommand[]` | `[]` | Pre-populated command history |
| `autoFocus` | `boolean` | `true` | Focus terminal on mount |
| `readOnly` | `boolean` | `false` | Disable command input |
| `loading` | `boolean` | `false` | Show loading indicator |
| `onCommand` | `(command: string) => Promise<string \| string[]> \| string \| string[]` | `undefined` | Command handler function |
