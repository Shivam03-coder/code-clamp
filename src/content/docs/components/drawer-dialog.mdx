## ðŸ”§ Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `React.ReactNode` | `undefined` | Drawer title |
| `description` | `React.ReactNode` | `undefined` | Drawer description |
| `children` | `React.ReactNode` | `required` | Drawer content |
| `triggerText` | `string` | `"Open Drawer"` | Text for the trigger button |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"right"` | Side from which the drawer appears |
| `open` | `boolean` | `undefined` | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | `undefined` | Function called when open state changes |
| `className` | `string` | `undefined` | Additional CSS classes for the drawer |
| `showCloseButton` | `boolean` | `true` | Whether to show the close button in the corner |

## ðŸŽ¨ Side Options

Choose from four sides to match your UI needs:
{/* 
\`\`\`tsx
// Right drawer (good for notifications, details panels)
<DrawerDialog side="right" />

// Left drawer (good for navigation, filters)
<DrawerDialog side="left" />

// Bottom drawer (good for mobile actions, keyboards)
<DrawerDialog side="bottom" />

// Top drawer (good for alerts, banners)
<DrawerDialog side="top" />
\`\`\` */}

The Drawer Dialog is perfect for creating sidebars, navigation menus, detail panels, or any situation where you need to display additional content without leaving the current page.
