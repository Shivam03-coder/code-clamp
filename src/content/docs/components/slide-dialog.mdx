## ðŸ”§ Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `React.ReactNode` | `undefined` | Dialog title |
| `description` | `React.ReactNode` | `undefined` | Dialog description |
| `children` | `React.ReactNode` | `required` | Dialog content |
| `triggerText` | `string` | `"Open Dialog"` | Text for the trigger button |
| `confirmText` | `string` | `"Confirm"` | Text for the confirm button |
| `cancelText` | `string` | `"Cancel"` | Text for the cancel button |
| `onConfirm` | `() => void` | `undefined` | Function called when confirm button is clicked |
| `showFooter` | `boolean` | `true` | Whether to show the dialog footer with buttons |
| `direction` | `"up" \| "down" \| "left" \| "right"` | `"up"` | Direction from which the dialog slides in |
| `className` | `string` | `undefined` | Additional CSS classes for the dialog |
| `duration` | `number` | `0.3` | Animation duration in seconds |

## ðŸŽ¨ Direction Options

Choose from four slide directions to match your UI context:

{/* \`\`\`tsx
// Slide up from bottom (good for notifications, toasts)
<SlideDialog direction="up" />

// Slide down from top (good for alerts, banners)
<SlideDialog direction="down" />

// Slide in from right (good for related content)
<SlideDialog direction="left" />

// Slide in from left (good for navigation, menus)
<SlideDialog direction="right" />
\`\`\` */}

The Slide Dialog is perfect for creating contextual animations that hint at where content is coming from or going to, providing better spatial awareness for users.
